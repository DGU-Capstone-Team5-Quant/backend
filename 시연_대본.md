- Questionary 인터랙티브 메뉴
- 실시간 거래 추천 결과는 구조화된 섹션으로 출력
- 차트는 matplotlib 기반, 자동으로 이미지 뷰어에서 열림
- **실제 데이터 기반**: 이 대본은 실제 백테스팅 결과와 실시간 분석 결과 반영

### 핵심 메시지:
1. **유연한 설정**: 빠른 백테스트(자동 설정)와 커스텀 백테스트(세부 조정) 모두 지원
2. **투명한 결과**: 수익(+0.68%)과 손실(-0.03%) 모두 있는 그대로 표시
3. **다층적 의사결정**: Bull/Bear/Trader/Manager의 4단계 협업 구조
4. **의견 충돌 관리**: Trader의 BUY 제안을 Manager가 SELL로 변경하는 리스크 관리
5. **메모리 기반 학습**: 과거 유사 패턴(-0.22% 수익률)을 참고한 의사결정.
6. **완전한 이력 관리**: 모든 결과를 JSON과 차트로 저장, 메모리 내보내기 지원.
